# Lumis - Gridea 博客主题

一款现代化、简洁优雅的 Gridea 博客主题，支持丰富的自定义功能和响应式设计。

## 主题特点

- **现代化设计** - 清新简约的界面风格，注重阅读体验
- **响应式布局** - 完美适配桌面端、移动端
- **全新Banner** - 全新设计Banner区域
- **搜索功能** - 支持文章模糊搜索
- **丰富的自定义配置** - 通过配置文件轻松定制主题

## 快速开始

1. 下载本主题到 Gridea 的 themes 目录
2. 在 Gridea 设置中选择该主题
3. 根据需要修改配置文件

### AI摘要配置

主题支持在文章页面显示AI生成的文章摘要卡片。

#### 使用方法

1. **在Gridea中编辑文章时**，在文章摘要（Abstract）字段填写AI生成的摘要内容
2. **配置AI摘要名称**：在主题设置中填写AI工具名称（如：DeepSeek、ChatGPT等），显示在摘要卡片右上角
3. **配置跳转链接**：填写AI工具链接，点击摘要卡片上的箭头可跳转到对应AI工具
